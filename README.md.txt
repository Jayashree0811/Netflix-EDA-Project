# Netflix EDA Project

This project performs exploratory data analysis on the Netflix Titles dataset using Python libraries like pandas, seaborn, and matplotlib.

## 📌 Goals:
- Compare Movies vs TV Shows
- Analyze release trends over the years
- Identify top countries and genres

## 📊 Tools Used:
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 🔗 Dataset:
[Netflix Titles - Kaggle](https://www.kaggle.com/datasets/shivamb/netflix-shows)

## 🖼️ Sample Visuals:
- TV Show vs Movie Distribution
- Top 10 Countries
- Content Added Over Time
- Popular Genres

---

## ✅ How to run:
Open the notebook file `Netflix_EDA.ipynb` in Jupyter or Google Colab and run all cells.
